# Contributing to KavachX 🛡️

First off, thank you for considering contributing to KavachX! It's people like you who make KavachX such a great tool for the AI community in Bharat and beyond.

## 🤝 Code of Conduct
By participating in this project, you agree to abide by our Code of Conduct (see `CODE_OF_CONDUCT.md`).

## 🚀 How Can I Contribute?

### Reporting Bugs
*   Check the [Issues](https://github.com/TheIndicSentinel/kavachxv2/issues) to see if the bug has already been reported.
*   If not, open a new issue using the **Bug Report** template.
*   Include as much detail as possible: steps to reproduce, environment details, and logs.

### Suggesting Enhancements
*   Open a new issue using the **Feature Request** template.
*   Explain the "Why" behind the feature and how it aligns with AI governance goals.

### Pull Requests
1.  Fork the repo and create your branch from `main`.
2.  If you've added code that should be tested, add tests.
3.  Ensure the test suite passes.
4.  Make sure your code follows the existing style guides.
5.  Write a clear, concise commit message.
6.  Open a PR against the `main` branch.

## 🛠️ Development Setup
See the [Setup Guide](docs/setup_guide.md) for detailed instructions on setting up your local environment.
